Symmetric

Mission Statement

Symmetric is a visual language that has been designed to represent the Outbrain brand and design voice of its digital products in a cohesive, consistent manner.

Symmetric is a live, up-to-date style guide and pattern library containing core reusable UI components maintained and created by the Outbrain UXD team.

The purpose of Symmetric is to be a reference guide that helps designers working on different products within Outbrain to maintain a consistent and unified voice, and to create designs that ultimately look like they belong to the same Outbrain family and are instantly recognizable as belonging to the overall Outbrain brand.

Initial exploration of Symmetric design patterns in the browser. Incorporating Outbrain's style guide into a live pattern library, in iteratable HTML.

Symmetric Values

These are the values that guided the creation of Symmetric, and the values that UXD uses to evaluate whether something makes it into the universal Symmetric language or not. These values also serve as a rubric - any changes made to Symmetric should be done with these values in mind. Values are either Aesthetic or Functional.

Aesthetic Values

  • Professional - Designs should showcase the fact that Outbrain is professional and enterprise software. Designs should reflect that Outbrain is one of the leading content marketing platforms in the industry.
  • Clean - Designs should strive to minimize clutter and superfluous elements that don't serve enough functional purpose, follow an organized grid, and contain consistent spacing.
  • Modern - Designs should be consistent with the latest UI patterns seen across the web.
  • Comprehensible/Legible - Designs should be as easy to read as possible. Proper whitespacing and contrast is important to us.
  • "Outbrainy" - Designs should be reflective of the Outbrain brand, which is professional yet playful and approachable (see: the Amelia logo).
  • Trustworthy/Intelligent - Designs should highlight data and data-driven insights.

What does the Symmetric Aesthetic look like?

  • Shared universal color palette
  • Complimentary colors to orange (brand) and product-specific hero colors
  • Modular grouping
  • Card designs where appropriate
  • Round corners on fields and modules
  • White space
  • Flat buttons
  • Padded Button/padded table rows feeling of space and mobile first approach
  • Consistent chart / device colors
  • Purposeful subtle micro animations and innovative user interaction design to support the flow of the experience and the response to user
  • Layout hierarchy that feels crisp and modern
  • Consistent typography and concrete labeling
  • Crisp and clean flat, solid-filled (not lined) iconography (provided by Material Design unless custom)

Functional Values

  • Up-to-Date - Anyone looking at Symmetric materials should be confident that whatever they are looking at is the latest version of Symmetric.
  • Efficiency/Reusability - Symmetric should make it easy for UI components to be re-used for people across the entire product team.
  • Consistency - Symmetric should encourage recognizable patterns, colors and elements that go hand-in-hand with reusability.
  • Scalability/Maintainability - creators should be able to change an element in one place and the change should propagate for all instances where that element is located.
  • Test-Driven/Adaptable

Symmetric should make the UX designers who create it feel:

  • Empowered - Do I feel like I can contribute to Symmetric?
  • Autonomous - Do I feel like Symmetric helps me as an individual designer on my projects, and that I can confidently work with it independent of my teammates?
  • Collaborative - Do I feel like Symmetric is a collaborative effort?
